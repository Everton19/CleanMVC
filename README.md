# CleanMVC
### This project was carried out using C#, ASP.NET, Entity Framework and PostgreSQL technologies on AWS to store product and category data.
### In the project I also used the concepts of Clean Architecture and tests were also done with XUnit to avoid errors and handle exceptions.
